# Approximate-Full-Adders
Parameterized versions of popular approximate adders designed with Verilog HDL or System Verilog.
***********************************Parameters******************************
N - No of bits
UPL - Upper Part Length
LPL - Lower Part Length
***********************************Modules*********************************
1. Precise Adder
2. Full Adder
3. Imprecise Adder
4. HOAANED
